Voting Counter

Voting Machine


I watched the American 2020 election, and became so fustrated at the way that "the greatest democracy in the world" were going about their elections during Covid 19.

With all the publicity on voter fraud, hacking of voting machines and allergations that people were voting with the dead peoples credentials. This made me wonder about creating a voting app that could beat the current Amercan system.

I started by making a voting machine using Tkinter as not to rely on web based tech, but rather a desktop app that was not connected to the internet. The sytem would show the candidates who were available for voting, tally votes and show a pie chart illustrating the percentages.

Two points that I would add to this project is a user database that uses personel credentials as well as facial recognition to identify that the voter is the correct person. I would also like to create a better security system to prevent code tampering.
